# AI Portfolio Platform

A production-grade AI engineering portfolio with RAG-powered chat, built with Next.js, FastAPI, and Supabase.

## Stack
- **Frontend**: Next.js 14, TypeScript, Tailwind CSS, Shadcn UI
- **Backend**: FastAPI, Python 3.11
- **Database**: Supabase PostgreSQL + pgvector
- **AI**: Gemini / Groq

## Structure
\`\`\`
apps/frontend    → Next.js app
apps/backend     → FastAPI server
packages/        → Shared types and config
\`\`\`
